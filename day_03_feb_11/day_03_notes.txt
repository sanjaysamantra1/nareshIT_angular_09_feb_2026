Datatypes 
=========
-Type of data we are dealing with


Primitive  :   number, string, boolean , undefined, bigint, symbol
non-primitive : object  {}  []

boolean : true and false are possible values
object  : {}  and [] are the possible values

Note: typeof operator is used to identify datatype of a value


primitve vs non-primitives
==========================
-Primitives are immutable , non-primitives are mutable
-Primitives are compared by value, non-primitves are compared by address


undefined vs null
==================
-when a variable is declared without any value, javscript assigns 'undefined' value 
-null is an intentional value assigned by developers
-typeof undefined is 'undefined' , typeof null is 'object'



Variables
=========
-A variable is a container to store data in memory.
ex: let name = 'Sanjay' 
    name  --> variable  , 'sanjay' ---> value stored in it

-Javascript provides 3 keywords to declare variables: 
    1. var
    2. let
    3. const
